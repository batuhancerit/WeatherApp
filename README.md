Weather App
A simple weather application built with React Native.
It provides real-time weather data, city search, geolocation support.

Features
-Location-based weather
-City search
-Real-time temperature
-Light / Dark mode support
-Component-based architecture

Technologies
-React Native
-Expo
-Axios
-Expo-Router
-react-native-async-storage
-OpenWeatherMap API

Installation
git clone https://github.com/USERNAME/WeatherApp.git
cd WeatherApp
npm install
npm start

Environment Setup
Create a .env file in the project root:
EXPO_PUBLIC_OPEN_WEATHER_API=YOUR_API_KEY
Get your API key from: https://openweathermap.org/api
